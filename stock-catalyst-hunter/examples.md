# 题材猎手示例

这些示例更适合 GitHub 浏览和快速理解 skill 用法。

## 示例 1

**输入**

```text
这条新闻利好哪些A股：美加墨世界杯赛程出炉，揭幕战将于6月12日3点举行。
```

**输出风格**

- 事件结论
- 主要受益方向
- 主要风险方向
- 股票表格
- 补充说明

## 示例 2

**输入**

```text
这消息对应什么题材股：中东冲突升级，国际油价盘中拉升。
```

**输出重点**

- 油气
- 航运
- 黄金避险

## 示例 3

**输入**

```text
帮我找这条消息的相关A股：日本可能限制部分半导体材料对华出口。
```

**输出重点**

- 光刻胶
- 电子材料
- 国产替代

## 示例 4

**输入**

```text
帮我看看这张图会影响哪些A股股票。
```

**适用场景**

- 财经截图
- 快讯截图
- 群聊转述
- 海报或新闻卡片
